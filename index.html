E-DEVLET E-İMZA

e-Devlet'e e-imzayla giriş

Tarayıcıda "e-imza ile giriş" dediğinizde açılan imza adımını Sello penceresi üstlenir: kartı okur, sözleşmeyi gösterir, PIN'inizle imzalar ve girişi tamamlar.

  • elektronik-imza.jar muadili — Java yok
  • Tanıdık arayüz, dağınık PIN tuş takımı
  • Kartla saniyeler içinde giriş
E-ADALET / UYAP

Tarayıcıda UYAP imzası

Sello arka planda sistem tepsisinde durur ve yerel bir imza köprüsü açar. e-Adalet/UYAP sayfalarındaki imza adımı kartınızla bu köprüden imzalanır — "Adalet E-imza" istemcisi gibi.

  • Resmî "Adalet E-imza" istemcisinin muadili
  • Yerel köprü (127.0.0.1) · yalnız onaylı UYAP alan adları
  • Resmî istemci kuruluysa çakışmamak için devre dışı kalır

Sık sorulanlar

Aklınızdaki sorular

Sello nedir?

e-Devlet'e e-imzayla giriş ile tarayıcıdaki e-Adalet (UYAP) kartlı imzayı tek uygulamada toplayan, Java gerektirmeyen, tek dosya bir Windows e-imza uygulamasıdır. Resmî elektronik-imza.jar ve "Adalet E-imza" istemcisinin hafif muadilidir.

e-Devlet e-imza için Java gerekir mi?

Sello ile hayır. Java, JVM veya Java Web Start kurmaya gerek yoktur; Windows'un yerleşik WebView2 motorunu kullanır ve çift tıkla saniyeler içinde açılır.

e-Adalet / UYAP'a kartla nasıl giriş yapılır?

Sello arka planda sistem tepsisinde çalışır ve yerel bir imza köprüsü açar. UYAP/e-Adalet sayfalarındaki e-imza adımı, kartınızla imzayı bu köprü üzerinden alır. Resmî "Adalet E-imza" istemcisi kuruluysa Sello köprüsü çakışmayı önlemek için devre dışı kalır.

Sello güvenli mi? PIN'im güvende mi?

İmza kartın içinde üretilir; kartın özel anahtarı dışarı çıkmaz. PIN loglanmaz, bellekte kısa tutulup silinir. TLS doğrulaması her zaman açıktır. Kart kilitliyse veya son deneme hakkındaysa giriş otomatik denenmez.

Başlangıçta açılmasını nasıl kapatırım veya kaldırırım?

Tepsi simgesine sağ tıklayıp Ayarlar'ı açın: "Windows ile başlat"ı kapatabilir veya "Sello'yu kaldır" ile autostart, yerel veri ve uygulamayı makineden tamamen kaldırabilirsiniz.

Hangi akıllı kartlar destekleniyor?

Sello PKCS#11 middleware ile çalışır; TÜBİTAK AKİS (akisp11.dll) doğrulanmıştır. Kart okuyucusu ve kart sürücüsü/middleware kurulu olmalıdır.

Hemen indir, çift tıkla başla

Windows 10/11 · akıllı kart okuyucusu ve middleware (AKİS için akisp11.dll) gerekir.
